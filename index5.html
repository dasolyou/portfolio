어바웃 미

발전하는 내가 되고자 끈기있게 공부하는 유다솔입니다. 1px의 차이도 예리하게 파악하고 사용자의 입장에서 꼼꼼하게 웹을 구성하겠습니다. 코드의 논리적 흐름을 분석하며 무한한 응용을 이루겠습니다. 상상하는 모든 웹을 실현하는 웹 퍼블리셔가 되겠습니다.
HTML 100% CSS 100% JavaScript 100% jQuery 100%

Web Standard
